# 🛒 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

The **E-Commerce Sales Analytics Dashboard** is an interactive Business Intelligence solution developed using Power BI to analyze and monitor online sales performance across multiple dimensions.

This dashboard provides a centralized analytical view of revenue, profit, customer behavior, product performance, and payment preferences. It enables dynamic filtering and drill-down capabilities to support data-driven decision-making.

---

## 🎯 Objective

The objective of this project is to transform raw e-commerce transactional data into actionable business insights by:

- Tracking key performance indicators (KPIs)
- Identifying high-performing states and product categories
- Analyzing monthly profit trends
- Understanding customer contribution and payment behavior
- Enabling quarter-wise interactive analysis

---

## 🛠️ Tech Stack

- 📊 **Power BI Desktop** – Dashboard development and visualization  
- 📂 **Power Query** – Data cleaning and transformation  
- 🧠 **DAX (Data Analysis Expressions)** – Creation of calculated measures and KPIs  
- 🔗 **Data Modeling** – Relationship building and cross-filtering  
- 📁 **File Format** – `.pbix` (development), `.png` (preview)

---

## 📂 Dataset Description

The dataset consists of structured e-commerce transactional data, including:

- Order Amount  
- Profit  
- Quantity Sold  
- Customer Name  
- Product Category & Sub-Category  
- Payment Mode  
- State  
- Monthly Sales Records  

The data was cleaned, transformed, and modeled to enable multi-dimensional analysis.

---

## 📊 Key Performance Indicators (KPIs)

- **Total Sales:** ₹438K  
- **Total Profit:** ₹37K  
- **Total Quantity Sold:** 5615  
- **Average Order Value (AOV):** ₹121K  

These KPIs provide a high-level summary of overall business performance.

## 📈 Dashboard Features & Visual Insights

### 🔹 State-wise Sales Analysis
Compares revenue contribution across different states such as Maharashtra, Madhya Pradesh, Uttar Pradesh, and Delhi.

### 🔹 Category-wise Quantity Distribution
Breakdown of product demand:
- Clothing (63%)
- Electronics (21%)
- Furniture (17%)

### 🔹 Monthly Profit Trend
Visualizes profit fluctuations across months to identify seasonal trends and low-performing periods.

### 🔹 Sub-Category Profit Analysis
Highlights top-performing product segments:
- Printers  
- Bookcases  
- Saree  
- Accessories  
- Tables  

### 🔹 Payment Mode Distribution
Analyzes customer payment preferences:
- COD (44%)  
- UPI (21%)  
- Debit Card (13%)  
- Credit Card (12%)  
- EMI (10%)

### 🔹 Customer-wise Revenue Contribution
Identifies key customers contributing significantly to total revenue.

### 🔹 Interactive Filters
- Quarter-based slicers (Q1–Q4)  
- Dynamic drill-down filtering  
- Category-level customization
